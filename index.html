ml-*{0-12} / pl-*{0-12} padding-left / / margin-left: number * 0.25rem;
mr-*{0-12} / pr-*{0-12} padding-right / margin-right: number * 0.25rem;

Transitions

You can send transitions-property property by using the class .transition-all

Class Properties
.transition-all all
.transition-colors color, background-color, border-color, opacity
